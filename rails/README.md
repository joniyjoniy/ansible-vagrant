# ansible-vagrant

## About
rails開発環境構築用playbook  

## Installation
- リポジトリとvagrantのセット
```
$ git clone https://github.com/joniyjoniy/ansible-vagrant.git
$ cd rails
```

- vagrantの起動
```
$ vagrant up
```

## Requirements
- vagrant
- VirtualBox
- ansible
